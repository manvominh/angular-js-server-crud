This is a CRUD web application front-end using angular version 17 connecting to api with js-server

There are sereral technical using in this web application:

  • angular 17
  • js-server

CRUD stands for main function :

  • C: Create User
  • R: Read User(s)
  • U: Update User
  • D: Delete User

This web application will be expanded in the future with more functions
